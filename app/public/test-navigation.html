Test du Menu Burger

Cette page permet de tester le menu burger en mode mobile.

  • Redimensionnez votre navigateur à moins de 768px de largeur
  • Cliquez sur le bouton burger (3 lignes) en haut à droite
  • Le menu devrait s'ouvrir avec une animation
  • Cliquez à nouveau pour fermer le menu

Instructions de test :

  1. Ouvrez les outils de développement (F12)
  2. Activez le mode responsive (icône mobile/tablette)
  3. Redimensionnez à une largeur mobile (ex: 375px)
  4. Cliquez sur le bouton burger
  5. Vérifiez que le menu s'ouvre et se ferme correctement
